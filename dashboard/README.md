# Dashboard

Web-based monitoring and analytics UI for SkySentinel.

## Features

- **Real-time Monitoring**: Live security event visualization
- **Threat Intelligence**: Interactive threat analysis
- **Compliance Dashboard**: Compliance status and reports
- **Resource Explorer**: Cloud resource inventory and relationships
- **Alert Management**: Security alert triage and response
- **Analytics**: Security metrics and trend analysis

## Technology Stack

- **Frontend**: React, TypeScript, TailwindCSS
- **Charts**: D3.js, Chart.js
- **State Management**: Redux Toolkit
- **API**: REST/GraphQL integration

## Installation

```bash
cd dashboard
npm install
npm start
```

## Access

- **Dashboard**: http://localhost:3000
- **API Docs**: http://localhost:3000/api/docs

## Components

- **Overview**: System health and summary
- **Events**: Event timeline and filtering
- **Threats**: Threat detection and analysis
- **Compliance**: Compliance framework tracking
- **Resources**: Resource inventory and relationships
- **Settings**: Configuration and user management
